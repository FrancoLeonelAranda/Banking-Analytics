\# Análisis Bancario (Proyecto Integral)



Proyecto integral de análisis de datos en el sector bancario, centrado en la comprensión de perfiles de clientes y riesgo crediticio.



\## Flujo de trabajo

1\. Datos almacenados en SQL Server

2\. Extracción y análisis en Python (pandas)

3\. Preparación para visualización BI

4\. Dashboard en Power BI (en curso)



\## Tecnologías

\- Python (pandas, matplotlib)

\- SQL Server

\- Jupyter Notebook

\- Power BI



\## Estado del proyecto

\- Base de datos creada

\- CSV importado

\- Conexión SQL–Python realizada

\- EDA inicial
